Push Ups

Muscle: Chest / Triceps / Core

  • How to: Plank position, elbows at ~45Â°, lower chest to floor, push back up.
  • Mistakes: Flaring elbows, sagging hips.
  • Sets/Reps: Beginner 3x8, Intermediate 4x12, Advanced 5x15-20.

Goblet Squat

Muscle: Legs / Glutes / Core

  • How to: Hold dumbbell at chest, sit back, knees track toes, drive through heels.
  • Mistakes: Heels lifting, collapsing knees.
  • Sets/Reps: Beginner 3x10, Intermediate 4x12, Advanced 4x15.

Bent-Over Row

Muscle: Back / Rear delts

  • How to: Hinge at hips, flat back, pull elbows toward hips.
  • Mistakes: Rounding back, jerking weight.
  • Sets/Reps: Beginner 3x8-10, Intermediate 4x10-12, Advanced 4x12-15.

Overhead Press

Muscle: Shoulders / Triceps

  • How to: Brace core, press dumbbells overhead, control down.
  • Mistakes: Overarching lower back.
  • Sets/Reps: Beginner 3x8, Intermediate 4x10, Advanced 4x10-12.

Lat Pulldown

Muscle: Lats / Upper back

  • How to: Pull bar to chest, elbows drive down, control on way up.
  • Mistakes: Leaning too far back, using momentum.
  • Sets/Reps: Beginner 3x10, Intermediate 4x10-12, Advanced 4x12-15.

Plank

Muscle: Core / Stability

  • How to: Elbows under shoulders, squeeze glutes, keep spine neutral.
  • Mistakes: Hips too high/low, holding breath.
  • Sets/Reps: Beginner 3x20s, Intermediate 3x40s, Advanced 4x60s.
